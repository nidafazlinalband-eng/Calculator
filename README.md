Project Overview

SCT_WD_2 is a fully functional and responsive Calculator Web Application designed to perform basic arithmetic operations.
The project focuses on DOM manipulation, event handling, keyboard input support, and error handling using pure HTML, CSS, and JavaScript.

The calculator allows users to perform operations through both on-screen buttons and keyboard keys, providing an intuitive and smooth user experience.
